# GPT-5.3 Codex 切替 — 叩く順（コピペ用）

**どこで叩くか:** 特記がなければ **ローカル Mac のターミナル**。  
**プロジェクトルートで:** `cd` で `anicca-project` に入ってから 3 と 4 を叩く。

---

## 1) VPS で gateway を止める

```bash
ssh anicca@46.225.70.241 'systemctl --user stop openclaw-gateway.service'
```

---

## 2) OAuth（ここだけ手動あり）

```bash
ssh -tt anicca@46.225.70.241 'openclaw models auth login --provider openai-codex'
```

- 出てきた **URL（https://auth.openai.com/...）** をブラウザで開く  
- ログインして許可  
- ブラウザが **http://127.0.0.1:1455/auth/callback?...** に飛んだら、その **URL 全体** をコピー  
- ターミナルに戻って **貼り付けて Enter**  
- 成功したら「2) 完了」として次へ

---

## 3) スクリプトを VPS に送る

**anicca-project のルートで実行:**

```bash
scp scripts/openclaw-vps/switch-to-gpt53codex.sh anicca@46.225.70.241:~/
```

---

## 4) VPS でスクリプトを実行

```bash
ssh anicca@46.225.70.241 'bash ~/switch-to-gpt53codex.sh'
```

- `OK: ログに 'agent model: openai-codex/gpt-5.3-codex' を確認しました` が出れば OK

---

## 5) Slack で確認

- @Anicca に **1回だけ** メンション  
- 返信が **1回** で、モデルが **openai-codex/gpt-5.3-codex** と出ていれば完了

---

**エラー時:**  
- 4 で「OAuth を完了してください」→ 2 をやり直してから 3・4 をもう一度  
- 「Unknown model」→ VPS で `openclaw models status --agent anicca` を叩き、`openai-codex/gpt-5.3-codex` が出ているか確認。出てなければ 2 をやり直し。
