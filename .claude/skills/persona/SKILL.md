# Anicca Target Persona

## Profile

| Item | Value |
|------|-------|
| Age | 25-35 |
| Region | Japan / English-speaking |
| Life stage | Working adult |

## Core Pain

**6-7年間、主体性の欠如と自己嫌悪のループから抜け出せていない。** 習慣アプリ10個以上試して全部3日坊主。

## Psychology

| Trait | Detail |
|-------|--------|
| Self-trust = 0 | 「自分との約束は守れない」が前提 |
| Resignation mode | 変わろうとすること自体が怖い |
| Hidden desire | 本当は変わりたい。でも言えない |
| Dependency | 誰かに引っ張ってほしい |

## Content/UI Judgment

| Effective Hook | Avoid |
|---------------|-------|
| 「6年間、何も変われなかった」 | 「簡単に習慣化！」（信じない） |
| 「習慣アプリ10個全部挫折」 | 「たった○日で！」（警戒する） |

**UI: 挫折を前提に、責めない、小さすぎるステップ。**
