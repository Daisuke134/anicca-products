### F1 後回しの理由

Free ユーザーの通知は **LOCAL notification**（`ProblemNotificationScheduler` が `Localizable.strings` から取得してスケジュール）。`Localizable.strings` はアプリにバンドルされるため、**App Store 再提出が必須**。ただし F1 は UX/収益に直接影響しないため 1.8.1 で対応。

Pro ユーザーの通知は `LLMNudgeService` → サーバーから取得するため再提出不要だが、F1 は Free ユーザー向け翻訳。

### F11 後回しの理由

7,092 文字列の追加は工数大。1.8.0 の A/B テスト + UX 改善にフォーカスし、1.8.1 で対応。

**CTA の trial period テキスト生成（動的）:**

```swift
/// StoreKit の introductoryDiscount から "1 week" / "3 days" / "1 month" を取得
private var trialPeriodText: String? {
    guard let discount = selectedPackage?.storeProduct.introductoryDiscount else { return nil }
    let period = discount.subscriptionPeriod
    switch period.unit {
    case .day:   return period.value == 1 ? "1 day" : "\(period.value) days"
    case .week:  return period.value == 1 ? "1 week" : "\(period.value) weeks"
    case .month: return period.value == 1 ? "1 month" : "\(period.value) months"
    case .year:  return period.value == 1 ? "1 year" : "\(period.value) years"
    @unknown default: return nil
    }
}
// JA の場合は Locale で切り替え
// "1 week" → "1週間", "3 days" → "3日間"
```
