### F9: RCPurchaseController.swift

**新規ファイル:** `aniccaios/aniccaios/Superwall/RCPurchaseController.swift`

```swift
import SuperwallKit
import RevenueCat
import StoreKit

enum PurchasingError: LocalizedError {
    case sk2ProductNotFound
    var errorDescription: String? { "StoreKit 2 product not found" }
}

final class RCPurchaseController: PurchaseController {
    func syncSubscriptionStatus() {
        Task {
            for await customerInfo in Purchases.shared.customerInfoStream {
                let entitlements = customerInfo.entitlements.activeInCurrentEnvironment.keys.map {
                    Entitlement(id: $0)
                }
                await MainActor.run {
                    Superwall.shared.subscriptionStatus = .active(Set(entitlements))
                }
            }
        }
    }

    func purchase(product: SuperwallKit.StoreProduct) async -> PurchaseResult {
        do {
            guard let sk2Product = product.sk2Product else {
                throw PurchasingError.sk2ProductNotFound
            }
            let rcProduct = RevenueCat.StoreProduct(sk2Product: sk2Product)
            let result = try await Purchases.shared.purchase(product: rcProduct)
            return result.userCancelled ? .cancelled : .purchased
        } catch let error as ErrorCode where error == .paymentPendingError {
            return .pending
        } catch {
            return .failed(error)
        }
    }

    func restorePurchases() async -> RestorationResult {
        do {
            _ = try await Purchases.shared.restorePurchases()
            return .restored
        } catch {
            return .failed(error)
        }
    }
}
```

### F11: Nudge バリエーション拡張

**目標:** 14日サイクル → 60日サイクル（2ヶ月繰り返しなし）

**ファイル:** `aniccaios/aniccaios/Models/ProblemType.swift`

```diff
  var notificationVariantCount: Int {
      switch self {
-     case .staying_up_late: return 21
-     default: return 14
+     case .staying_up_late: return 60
+     default: return 60
      }
  }
```

**コンテンツ生成方法:**

```
Step 1: 問題タイプごとに +46 バリエーション（staying_up_late は +39）を EN で作成
  → 既存コンテンツのトーン・長さ・スタイルを踏襲
  → 5つの角度をローテーション:
    ① 動機付け（"You can do this"）
    ② 共感（"We know it hurts"）
    ③ 行動指示（"Put the phone down now"）
    ④ 問いかけ（"What would tomorrow-you want?"）
    ⑤ 名言/仏教引用（"Pain is inevitable, suffering is optional"）

Step 2: JA 翻訳（同じトーンで自然な日本語）
Step 3: ES 翻訳
Step 4: DE/FR/pt-BR 翻訳

キー命名: nudge_{problem}_notification_{N} / nudge_{problem}_detail_{N}
  staying_up_late: N=22〜60（既存 1〜21）
  他12問題: N=15〜60（既存 1〜14）
```

**追加文字列数:**

| 対象 | 計算 | 数 |
|------|------|-----|
| staying_up_late 追加 | 39 × 2 (notification+detail) | 78 |
| 他12問題 追加 | 12 × 46 × 2 | 1,104 |
| EN 合計 | | 1,182 |
| × 6言語 | 1,182 × 6 | **7,092 文字列** |

**実行手順:**
1. EN で 1,182 文字列を生成（LLM で既存スタイル踏襲）
2. JA/ES/DE/FR/pt-BR に翻訳
3. 各言語の Localizable.strings に追加
4. ProblemType.notificationVariantCount を 60 に変更

```

  - Platform: iOS
  - Public API Key 取得

  - anicca_monthly: $9.99/月
  - anicca_annual: $49.99/年
Step 4: MCP で Paywall テンプレート作成
  - Variant A: 現在のデザイン再現
  - Variant B: CRO 最適化デザイン（paywall-upgrade-cro BP 適用）
Step 5: MCP で Campaign 作成
  - placement: onboarding_paywall
  - audience: All Users
  - experiment: 50% A / 50% B
Step 6: MCP で Campaign 作成（upgrade）
  - placement: upgrade
  - 同じ experiment 設定
```
