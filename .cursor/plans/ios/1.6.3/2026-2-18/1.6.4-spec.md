## 1. 概要（What & Why）

App Review 承認済み（1.6.3）を受け、1.6.4 で以下の 4 点を修正・改善する。

| # | 機能 | 理由 |
|---|------|------|
| F1 | 通知タイトル ローカライズ修正 | ES/DE/FR/pt-BR の `problem_*_notification_title` が全て英語 → デバイスをドイツ語/スペイン語等にしたユーザーが英語タイトルの通知を受け取っている |
| F2 | 問題カード スワイプ削除 | ユーザーが追加した問題（ProblemType）を削除できない。習慣を変えたとき行き場がない → 離脱につながる |
| F3 | メイン画面タイトル変更 | `tab_mypath` = "My Path" / "マイパス" → "Anicca" / "アニッチャ"。ブランド統一 |
| F4 | App Store 日本語アプリ名変更 | App Store 日本語ページ + デバイスホーム画面名が "Anicca" → "アニッチャ" に変更。日本語ユーザーに馴染みやすくする |

### F4: App Store 日本語アプリ名変更

**As-Is:**
- デバイスホーム画面（JA）: "Anicca"
- App Store 日本語ページ名: "Anicca"

**To-Be:**
- デバイスホーム画面（JA）: "アニッチャ"
- App Store 日本語ページ名: "アニッチャ"

**変更 1: InfoPlist.strings（デバイス表示名）**

```
// ja.lproj/InfoPlist.strings に追加
"CFBundleDisplayName" = "アニッチャ";
```

**変更 2: App Store Connect（ストア表示名）**

`asc` CLI で日本語ロケールの App Name を更新する。

```bash
# Japanese App Store name を アニッチャ に更新
asc apps list --bundle-id com.anicca.ios
# → app ID 取得

asc versions list --app-id <ID> --platform IOS --filter[appStoreState]=PREPARE_FOR_SUBMISSION
# → version ID 取得

asc localizations update --version-id <VERSION_ID> --locale ja --name "アニッチャ"
```

注意: ASC の `name` フィールドは App Store のタイトル表示名。変更は次回提出時に反映。
