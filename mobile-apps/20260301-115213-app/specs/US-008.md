## Steps

### Phase A: スクショ撮影（screenshot-creator Step 0）
1. rm -rf screenshots/ docs/screenshots/
2. xcrun simctl でシミュレータ撮影（4枚、5分以内）:
   - iPhone SE 3rd gen 推奨（Dynamic Island なし）
   - mkdir -p screenshots/raw
   - screen_1.png (Home), screen_2.png (主要機能), screen_3.png (別の機能)
   - **paywall.png (ペイウォール画面 — サブスク Review 用に必須)**

### Phase B: プロモスクショ作成（screenshot-creator Step 1-7 = Pencil MCP）
3. Step 1: アプリ情報を progress.txt から取得
4. Step 2: Pencil MCP get_style_guide でスタイルガイド取得
5. Step 3: ヘッドライン + サブテキスト作成（14文字以内）
6. Step 4: Pencil MCP I() でフレーム構築（3枚）
   - CaptionArea + MockupArea（実スクショ image fill）
   - layout: "vertical" 必須、width: "fill_container" 必須、fontSize 最大 22px
7. Step 5: batch_get + snapshot_layout バリデーション（10項目）
8. Step 6: get_screenshot 品質レビュー（7/10 以上）
9. Step 7: mkdir -p screenshots/framed && sips -z 2796 1290 各ファイル

### Phase C: iPad スクショ（submission-checklist E4b — Submit 必須）
10. iPhone スクショを sips でリサイズ → screenshots/framed-ipad/ (2048×2732)
    - asc screenshots upload --device-type APP_IPAD_PRO_3GEN_129

### Phase D: ASC アップロード + メタデータ
11. iPhone スクショ 3枚アップロード（en-US + ja）→ APP_IPHONE_67
12. iPad スクショ 3枚アップロード（en-US + ja）→ APP_IPAD_PRO_3GEN_129
13. メタデータ同期（en-US + ja）: description, keywords, whatsNew

### Phase E: ビルド
14. security unlock-keychain -p "$KEYCHAIN_PASSWORD"
15. xcodebuild clean archive → exportArchive → .ipa
16. asc builds upload → processingState=VALID 待ち

### Phase F: 提出前設定（submission-checklist D/E 全項目）
17. copyright 設定 (E7)
18. contentRightsDeclaration = DOES_NOT_USE_THIRD_PARTY_CONTENT (E8)
19. app pricing 設定 (E9)
20. usesIdfa = false (E11)
21. primaryCategory 設定 (E12)
22. Age Rating 22項目
23. Review Details (--demo-account-required false)
24. Availability (175 territories)
25. **サブスク Review Screenshot 添付 (D6/D7)**:
    - asc subscriptions review-screenshots create --subscription-id 6759877016 --file screenshots/raw/paywall.png
    - asc subscriptions review-screenshots create --subscription-id 6759877119 --file screenshots/raw/paywall.png
    - ⚠️ ペイウォール画面の RAW スクショ（プロモ加工なし）。Apple レビュアー確認用。
26. サブスク ローカライゼーション確認 (D8/D9)

### Phase G: 報告
27. Slack: "⏸️ App Privacy を ASC Web で設定してください"
