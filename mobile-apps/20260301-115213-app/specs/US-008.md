## Steps

### Phase A: ビルド + インストール + ATTダイアログ処理
1. シミュレータ選択（Dynamic Island なし推奨）:
   ```bash
   UDID=$(xcrun simctl list devices booted | grep -oE '[A-F0-9-]{36}' | head -1)
   ```
2. クリーンビルド + インストール:
   ```bash
   xcodebuild clean build -project MicroMoodiOS/MicroMoodiOS.xcodeproj \
     -scheme MicroMoodiOS -destination "platform=iOS Simulator,id=$UDID" \
     -derivedDataPath DerivedData CODE_SIGN_IDENTITY=-
   xcrun simctl uninstall "$UDID" <bundle_id> 2>/dev/null
   xcrun simctl install "$UDID" "DerivedData/Build/Products/Debug-iphonesimulator/MicroMoodiOS.app"
   xcrun simctl launch "$UDID" <bundle_id>
   sleep 5
   ```
3. ATTダイアログ処理（出てたら dismiss）:
   ```bash
   axe describe-ui --udid "$UDID" 2>&1 | grep -q "トラッキング" && \
     axe tap --label "アプリにトラッキングしないように要求" --udid "$UDID"
   sleep 2
   ```

### Phase B: AXe で画面遷移 + スクショ撮影（実証済み 2026-03-02）
4. Tab Bar 座標取得:
   ```bash
   # axe describe-ui で Tab Bar の frame を確認
   axe describe-ui --udid "$UDID" | python3 -c "
   import json,sys
   data=json.load(sys.stdin)
   def find(n):
     if 'Tab Bar' in (n.get('AXLabel') or ''):
       f=n['frame']; print(f'TAB_Y={f[\"y\"]+f[\"height\"]/2}'); print(f'TAB_W={f[\"width\"]}')
     for c in n.get('children',[]): find(c)
   for d in (data if isinstance(data,list) else [data]): find(d)
   "
   # 4 tabs: x = TAB_W/8, 3*TAB_W/8, 5*TAB_W/8, 7*TAB_W/8
   ```
5. スクショ撮影（5枚、全て異なる画面）:
   ```bash
   mkdir -p screenshots/raw

   # Screen 1: Today (Tab 1)
   axe tap -x $TAB1_X -y $TAB_Y --udid "$UDID" && sleep 1
   axe screenshot --output screenshots/raw/screen_1_home.png --udid "$UDID"

   # Screen 2: History (Tab 2)
   axe tap -x $TAB2_X -y $TAB_Y --udid "$UDID" && sleep 1
   axe screenshot --output screenshots/raw/screen_2_history.png --udid "$UDID"

   # Screen 3: Insights (Tab 3)
   axe tap -x $TAB3_X -y $TAB_Y --udid "$UDID" && sleep 1
   axe screenshot --output screenshots/raw/screen_3_insights.png --udid "$UDID"

   # Screen 4: Settings (Tab 4)
   axe tap -x $TAB4_X -y $TAB_Y --udid "$UDID" && sleep 1
   axe screenshot --output screenshots/raw/screen_4_settings.png --udid "$UDID"

   # Screen 5: Paywall (tap "Upgrade to Pro")
   axe tap --label "Upgrade to Pro" --udid "$UDID" && sleep 3
   axe screenshot --output screenshots/raw/screen_5_paywall.png --udid "$UDID"
   ```

### Phase C: Koubou フレーム追加
6. ```bash
   pip install koubou==0.13.0 2>/dev/null
   mkdir -p screenshots/framed
   DEVICE=$(asc screenshots list-frame-devices --output json | python3 -c "import json,sys;print(json.load(sys.stdin)[0]['id'])")
   for f in screenshots/raw/screen_*.png; do
     asc screenshots frame --input "$f" --output-dir screenshots/framed --device "$DEVICE" --output json
   done
   ```

### Phase D: iPhone 5.5" + iPad リサイズ
7. ```bash
   mkdir -p screenshots/framed-55 screenshots/framed-ipad
   for f in screenshots/framed/*.png; do
     base=$(basename "$f")
     sips -z 2208 1242 "$f" --out "screenshots/framed-55/$base"
     sips -z 2732 2048 "$f" --out "screenshots/framed-ipad/$base"
   done
   ```
