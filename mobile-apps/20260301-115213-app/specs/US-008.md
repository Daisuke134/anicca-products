## Steps

### Phase A: スクリーンショット撮影 + プロモ化（screenshot-creator SKILL.md に完全に従え）
1. FIRST: rm -rf screenshots/ docs/screenshots/
2. **Step 0 (screenshot-creator):** xcrun simctl でシミュレータ撮影（3枚、5分以内）
   - xcrun simctl list devices available | grep iPhone
   - xcrun simctl boot <UDID>
   - xcrun simctl install <UDID> /path/to/App.app
   - xcrun simctl launch <UDID> <BUNDLE_ID>
   - xcrun simctl io <UDID> screenshot screenshots/raw/screen_1.png
   - 3枚の異なる画面（Home、主要機能、Paywall等）
3. **Step 1-3 (screenshot-creator):** ヒアリング情報をprogress.txtから取得 → スタイルガイド取得 → コピー作成
4. **Step 4 (screenshot-creator):** Pencil MCP でプロモデザイン構築
   - Pencil MCP の I() 関数でフレーム・テキスト・モックアップを構築
   - 実スクショを PhoneMockup の image fill として使用
   - get_screenshot で目視確認
5. **Step 5-6 (screenshot-creator):** バリデーション + 品質レビュー
6. **Step 7 (screenshot-creator):** PNG エクスポート + sips リサイズ（1290×2796）
   ```
   sips -z 2796 1290 screenshots/framed/screen_1.png
   sips -z 2796 1290 screenshots/framed/screen_2.png
   sips -z 2796 1290 screenshots/framed/screen_3.png
   ```

### Phase B: ASC アップロード + メタデータ
7. ASC にスクショアップロード（en-US + ja）:
   - asc screenshots upload --version-localization <LOC_ID> --path screenshots/framed/ --device-type APP_IPHONE_67
8. メタデータ同期（en-US + ja）— app-description-writer + keyword-optimizer に従う
9. Archive + .ipa export:
   ```
   source ~/.config/mobileapp-builder/.env
   security unlock-keychain -p "$KEYCHAIN_PASSWORD" ~/Library/Keychains/login.keychain-db
   ```
   asc-xcode-build SKILL.md に従う
10. Upload build, attach to version
11. Age Rating (all 22 items), Review Details (--demo-account-required false), Availability (175 territories), Content Rights (DOES_NOT_USE_THIRD_PARTY_CONTENT)

### Phase C: サブスク Review スクショ
12. サブスク Review スクリーンショット添付（Apple §2.1 拒否防止）:
    - asc subscriptions review-screenshots create --subscription-id <MONTHLY_ID> --file screenshots/raw/paywall.png
    - asc subscriptions review-screenshots create --subscription-id <ANNUAL_ID> --file screenshots/raw/paywall.png

### Phase D: 報告
13. Slack 報告: "⏸️ App Privacy を ASC Web で設定してください: https://appstoreconnect.apple.com/apps/$APP_ID/distribution/privacy"
