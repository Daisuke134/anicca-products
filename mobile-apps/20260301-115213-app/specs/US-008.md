## Steps

### Phase A: シミュレータ起動 + アプリインストール
1. iPhone SE 3rd gen を起動:
   ```bash
   SE_UDID=$(xcrun simctl list devices | grep "SE (3rd" | grep -oE '[A-F0-9-]{36}' | head -1)
   xcrun simctl boot "$SE_UDID" 2>/dev/null || true
   ```
2. アプリビルド + インストール:
   ```bash
   xcodebuild build -scheme MicroMoodiOS -destination "id=$SE_UDID" -derivedDataPath DerivedData
   APP_PATH=$(find DerivedData -name "*.app" -path "*/Debug-iphonesimulator/*" | head -1)
   xcrun simctl install "$SE_UDID" "$APP_PATH"
   xcrun simctl launch "$SE_UDID" com.anicca.micromood
   sleep 5
   ```

### Phase B: 画面遷移 + スクショ撮影（4枚、全て異なる画面）
3. mkdir -p screenshots/raw
4. **Screen 1 — Home画面（ムード入力）:**
   ```bash
   xcrun simctl io "$SE_UDID" screenshot screenshots/raw/screen_1.png
   ```
5. **Screen 2 — History画面:**
   AXe またはタブバータップで遷移:
   ```bash
   # AXe が使えない場合: deep link or simctl openurl
   xcrun simctl openurl "$SE_UDID" "micromood://history" 2>/dev/null || true
   sleep 2
   xcrun simctl io "$SE_UDID" screenshot screenshots/raw/screen_2.png
   ```
   AXe が使えない + deep link もない場合:
   ```bash
   # xcrun simctl で座標タップ（History タブ = 画面下部左から2番目）
   # iPhone SE 3rd gen: 375x667pt, タブバー y≈635, History x≈140
   xcrun simctl io "$SE_UDID" tap 140 635 2>/dev/null || true
   sleep 2
   xcrun simctl io "$SE_UDID" screenshot screenshots/raw/screen_2.png
   ```
6. **Screen 3 — Insights画面:**
   ```bash
   # Insights タブ = 左から3番目, x≈235
   xcrun simctl io "$SE_UDID" tap 235 635 2>/dev/null || true
   sleep 2
   xcrun simctl io "$SE_UDID" screenshot screenshots/raw/screen_3.png
   ```
7. **Screen 4 — Paywall画面（サブスクReview用）:**
   ```bash
   # Settings タブ → Upgrade to Pro ボタン
   xcrun simctl io "$SE_UDID" tap 330 635 2>/dev/null || true
   sleep 2
   # Upgrade to Pro ボタンをタップ（Settings画面上部付近）
   xcrun simctl io "$SE_UDID" tap 187 150 2>/dev/null || true
   sleep 3
   xcrun simctl io "$SE_UDID" screenshot screenshots/raw/paywall.png
   ```

### Phase C: Koubou フレーム追加
8. ```bash
   mkdir -p screenshots/framed
   for f in screenshots/raw/screen_*.png; do
     asc screenshots frame --input "$f" --output-dir screenshots/framed --device iphone-se-3
   done
   ```

### Phase D: iPhone 5.5" + iPad リサイズ
9. ```bash
   mkdir -p screenshots/framed-55 screenshots/framed-ipad
   for f in screenshots/framed/screen_*.png; do
     base=$(basename "$f")
     sips -z 2208 1242 "$f" --out "screenshots/framed-55/$base"
     sips -z 2732 2048 "$f" --out "screenshots/framed-ipad/$base"
   done
   ```

### Phase E: ASC アップロード + メタデータ
10. スクショアップロード（en-US + ja）:
    ```bash
    source ~/.config/mobileapp-builder/.env
    for f in screenshots/framed/screen_*.png; do
      asc screenshots upload --app $APP_ID --locale en-US --device-type APP_IPHONE_67 --file "$f"
      asc screenshots upload --app $APP_ID --locale ja --device-type APP_IPHONE_67 --file "$f"
    done
    # 5.5"
    for f in screenshots/framed-55/screen_*.png; do
      asc screenshots upload --app $APP_ID --locale en-US --device-type APP_IPHONE_55 --file "$f"
      asc screenshots upload --app $APP_ID --locale ja --device-type APP_IPHONE_55 --file "$f"
    done
    # iPad
    for f in screenshots/framed-ipad/screen_*.png; do
      asc screenshots upload --app $APP_ID --locale en-US --device-type APP_IPAD_PRO_3GEN_129 --file "$f"
      asc screenshots upload --app $APP_ID --locale ja --device-type APP_IPAD_PRO_3GEN_129 --file "$f"
    done
    ```
11. メタデータ同期（en-US + ja）: description, keywords, whatsNew

### Phase F: ビルド
12. ```bash
    security unlock-keychain -p "$KEYCHAIN_PASSWORD" ~/Library/Keychains/login.keychain-db
    ```
13. xcodebuild clean archive → exportArchive → .ipa
14. asc builds upload → processingState=VALID 待ち

### Phase G: 提出前設定
15. copyright, contentRightsDeclaration, pricing, usesIdfa, primaryCategory
16. Age Rating, Review Details, Availability (175 territories)
17. **サブスク Review Screenshot 添付:**
    ```bash
    asc subscriptions review-screenshots create --subscription-id $MONTHLY_SUB_ID --file screenshots/raw/paywall.png
    asc subscriptions review-screenshots create --subscription-id $ANNUAL_SUB_ID --file screenshots/raw/paywall.png
    ```

### Phase H: 報告
18. Slack: "⏸️ App Privacy を ASC Web で設定してください"
