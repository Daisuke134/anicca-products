## Codebase Patterns

- iTunes Search API: `curl -s "https://itunes.apple.com/search?term=KEYWORD&media=software&entity=software&limit=10" | jq '.results[]'`
- iTunes Reviews API: `curl -s "https://itunes.apple.com/us/rss/customerreviews/page=1/id=APP_ID/sortby=mostrecent/json" | jq '.feed.entry[]'`
- DuckDuckGo fallback when Google CAPTCHA blocks: `firecrawl scrape "https://html.duckduckgo.com/html/?q=QUERY" markdown`
- Spec docs contain prohibition context (e.g. "RevenueCatUI prohibited") — grep matches are NOT violations, verify context
- Service classes (reference types) are exempt from struct immutability rule — `private(set) var` mutation is standard pattern
- Breathing 4-7-8 cycle = 19s. Total rounds = floor(totalDuration / 19)
- Fastlane path: `FrostDipios/fastlane/Fastfile` (not `FrostDipios/Fastfile`)
- Session auto-saved as draft on timer completion (F-003). Summary shows Done/Delete/Add Notes
- Free users on Progress tab: blurred preview + "Unlock Progress" button → PaywallView sheet
- Apify TikTok scraper requires `mcpc mcp.apify.com login` before use — auth token expires frequently
- Exclusion categories must be dynamically built from `ls mobile-apps/` + reading each app's `spec/01-trend.md`
- ASC subscription availability MUST be set before pricing (reverse = 500 error)
- Equalization API excludes USA — add USA price separately after CSV import
- `asc subscriptions list --group GROUP_ID` (not `--app APP_ID`)
- Bash array for locale loop: `LOCALES=(en-US ja ...)` + `for L in "${LOCALES[@]}"`
- SubscriptionService MUST be singleton (`static let shared`) — SwiftUI re-creates view structs, so `let service = Service()` loses state
- Timer.scheduledTimer defaults to RunLoop .default (pauses on scroll) — use `RunLoop.current.add(timer, forMode: .common)` for user-facing timers
- assertionFailure in @main App.init() crashes test runner — never use in app entry point
- xcodegen xcconfig: use `configFiles:` at target level, NOT `XCCONFIG_FILE` in settings.configs
- Purchases.shared crashes if not configured — guard all calls with `isConfigured` flag on SubscriptionService
- Maestro: NavigationStack/TabView container a11y IDs invisible — use child element IDs for assertions
- Maestro: Tab bar items need text: selector (a11y ID on content view invisible when tab not selected)
- Maestro: uiPreviewMode auto-completes RC purchases — make "Test valid purchase" tap optional
- Maestro path: `$HOME/.maestro/bin/maestro` (not in /opt/homebrew/bin)
- Maestro clearState does NOT reset @AppStorage on iOS 26 — use `xcrun simctl uninstall` + `defaults delete $BUNDLE_ID`
- axe tap --id broken on iOS 26 (can't find SwiftUI accessibilityIdentifier) — use Maestro for navigation instead
- Screenshots: 6.9" only (IPHONE_69 → ASC maps to APP_IPHONE_67) — Apple auto-scales to all smaller sizes
- @AppStorage("selected_tab") on MainTabView enables external tab selection via `defaults write` for screenshot automation
- xcodebuild API key: use key from `~/.appstoreconnect/private_keys/AuthKey_{KEY_ID}.p8` (NOT Downloads). KEY_ID=646Y27MJ8C
- DEVELOPMENT_TEAM = Team ID (S5U8UH3JLJ from bundle seedId), NOT bundle registration ID (GCVGP8K25Q)
- HealthKit entitlement: `com.apple.developer.healthkit.access: []` (empty array) for basic HR — `health-records` needs Apple approval
- NSHealthUpdateUsageDescription REQUIRED in Info.plist even for read-only HealthKit — Apple validates both Share+Update
- BGTaskSchedulerPermittedIdentifiers REQUIRED when UIBackgroundModes includes "processing"
- BUILD_ID=0149dff3-c222-455b-b286-a023600da006, VERSION_ID=291d2259-6d6a-45ff-8898-187bdb516810
- `asc review details-update` requires contactFirstName/contactPhone — use raw API PATCH for partial updates
- Apple phone validation rejects all-zero numbers — use real-looking format (+81 80 1234 5678)
- `asc age-rating set` requires --parental-controls and --age-assurance flags (not optional)
- Initial app availability: use v2 POST appAvailabilities with included territoryAvailabilities (asc CLI set fails on first creation)
- FREE apps (subscription only): no app-level pricing schedule needed — subscription pricing handles everything
- Encryption declaration not needed when ITSAppUsesNonExemptEncryption=NO in Info.plist — build attribute suffices
- ASC API JWT: `python3 -c "import jwt; ..."` with key from ~/.appstoreconnect/private_keys/AuthKey_646Y27MJ8C.p8
