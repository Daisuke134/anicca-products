## Codebase Patterns

| VStack a11y ID hides children | `.accessibilityIdentifier()` on a VStack/container makes Maestro unable to find child elements. Put IDs on leaf views only, or use child element IDs for assertions |
| JP locale notification dialog | シミュレータが日本語ロケールの場合、通知許可ダイアログのボタンは「許可」。Maestro で `text: "許可\|Allow"` + `optional: true` を使う |
| Stale system dialog after clearState | `clearState` は iOS システムダイアログ（通知許可等）をクリアしない。launchApp 直後に `tapOn: "許可\|Allow" optional: true` で残留ダイアログを消す |
| RC_PUBLIC_KEY must be set in xcconfig | Config.xcconfig の `RC_PUBLIC_KEY` が空だと `Purchases.configure()` がスキップされ、paywall で `Purchases has not been configured` クラッシュ |
| uiPreviewMode purchase dialog | `uiPreviewMode: true` では RC 購入ダイアログ（"Test valid purchase"）が表示されない場合がある。`optional: true` にする |
| NavigationView toolbar iOS 26 | iOS 26 では `NavigationView` + `.toolbar` の `ToolbarItem` が表示されない場合がある。VStack 内の HStack + Button に変更 |
| maestro PATH | maestro CLI は `/Users/anicca/.maestro/bin/maestro` にある。PATH に追加必須 |

| Package unit test limitation | RevenueCat `Package` is a final class — cannot be instantiated in unit tests. Purchase flow with real packages tested in E2E (US-007) |
| Onboarding hasCompletedOnboarding | `@AppStorage("hasCompletedOnboarding")` controls onboarding/timer routing in ContentView. All apps use same key (US-008a screenshot dependency) |

| パターン | 詳細 |
|---------|------|
| Apify token ロード | `source .env` では APIFY_TOKEN が空になる。`eval "$(grep '^APIFY_TOKEN=' ~/.config/mobileapp-builder/.env)"` を使うこと |
| Apify 非同期実行 | POST runs → 待機 → GET actor-runs/{id} で SUCCEEDED → GET datasets/{id}/items |
| iTunes 低評価レビュー | `https://itunes.apple.com/us/rss/customerreviews/page=1/id=APP_ID/sortby=mostrecent/json` + python3 フィルタ |

| Gate check false positive | grep for Rule violations (RevenueCatUI, Mixpanel, Analytics) matches prohibition text in docs. These are not actual violations — they describe what is prohibited |
| Doc generation order | ARCHITECTURE → DESIGN_SYSTEM → UX_SPEC (strict). UX_SPEC references DESIGN_SYSTEM color tokens |

| Phase table ↔ Section heading | IMPL_GUIDE の Phase table ラベルと実際のセクション見出しを一致させること。Phase 4 に対応するセクションが欠落していた |
| ObservableObject mutation | SwiftUI @Published properties は immutability rule の例外。ARCHITECTURE.md §13 に記載 |
| Dynamic Type exception | Timer display (56pt) は固定サイズ。DESIGN_SYSTEM.md に例外として記載が必要 |
| apps create requires 2FA | `asc apps create` は iris session があっても独自に Apple ID 認証する。`--two-factor-code` 必須。API Key 認証不可 |
| source .env in Bash tool | `source ~/.config/.../projects/<slug>/.env` は Bash tool 内で変数が展開されない場合あり。`export VAR=VALUE` で直接設定するか、`cat .env` で値を確認してからコマンドに埋め込むこと |
| availability before pricing | ASC API で pricing 設定前に availability 設定が必須。逆だと 500 エラー |
| equalization excludes USA | `price-points equalizations` は USA を含まない(174カ国)。USA は `prices add` で別途追加が必要 |
| 3-plan subscription | RevenueCat SOSA 2025: weekly が47% of revenue。3プラン(weekly+monthly+annual)必須 |
| ActivityKit iOS 16.2+ | `Activity.request`, `activity.end`, `activity.update(ActivityContent)` は iOS 16.2+。`if #available(iOS 16.2, *)` 必須 |
| Shared sources for Widget Extension | EyeBreakAttributes は App + WidgetExtension 両方で使用。`Shared/` ディレクトリに配置し project.yml で両ターゲットに追加 |
| xcodegen shared sources | project.yml の sources に `- path: Shared` を追加して複数ターゲットでコード共有 |
| iPhone 17 Pro simulator | Xcode 26.2 では iPhone 16 Pro シミュレータなし。iPhone 17 Pro を使用 |
| @MainActor test classes | @MainActor ViewModel のテストクラスには `@MainActor` アノテーション必須。なければ isolation error |
| EnvironmentObject for services | TimerService/SubscriptionService は @EnvironmentObject で注入。View 内の ViewModel は StateObject の軽量コンテナのみ |
| DesignSystem directory | AppColors, AppTypography, AppSpacing, AppCornerRadius, AppAnimations は `DesignSystem/` に集約。Components/ にPrimaryButton等 |
| Test target RevenueCat dependency | テストターゲットで RevenueCat.Package を使うには project.yml に RevenueCat パッケージ依存を追加必須 |
| PrimaryButton reuse | OnboardingView の Next/Allow Notifications ボタンは PrimaryButton コンポーネントを直接使う。手動で Button を組むより統一的 |
| PackageCardView a11y mapping | 3-plan model (weekly/monthly/annual) では packageType ベースで a11y ID を振り分ける。isAnnual bool だけでは不十分 |
| NSLocalizedString for PrimaryButton | PrimaryButton の title は String 型。LocalizedStringKey ではなく NSLocalizedString() でラップして渡す |
| RC configure single source | RevenueCat Purchases.configure() は1箇所のみ。App struct ではなく SubscriptionService.configure() に集約。重複呼び出し禁止 |
| Localization key naming | dot-notation keys (e.g. timer.status.ready) を使い、en + ja 両方を xcstrings に明示記載。暗黙のキー＝値パターン禁止 |
