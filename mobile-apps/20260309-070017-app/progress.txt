
## Codebase Patterns

| パターン | 詳細 |
|--------|------|
| fastlane test | `LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 fastlane test` 必須（xcpretty UTF-8 バグ回避） |
| xcodegen | ファイル追加後は `xcodegen generate` 必須（テストファイル含め自動pick-up） |
| SubscriptionService env | `@Environment(SubscriptionService.self)` で取得。`SubscriptionService()` 新規生成禁止 |
| URL optional | `URL(string:)!` 禁止。`if let url = URL(string:)` で optional binding |
| Background Timer | `startDate = Date.now` で計時開始。タイマーTick: `Int(Date.now.timeIntervalSince(startDate))`。suspend後も正確 |
| Greenlight | `greenlight preflight <dir>` (xcodeproj じゃなくディレクトリを渡す) |
| Maestro sequential | `maestro/config.yaml` に `executionOrder.flowsOrder` を設定 → sequential実行 |
| Maestro btn off-screen | `tapOn: id: "btn_maybe_later"` の前に `- scroll` 必須。ボタンがfold下にある場合、Maestroの自動スクロールは動作しない |
| PaywallView完了 | `@AppStorage("hasCompletedOnboarding")` をPaywallView直接に持ち、`hasCompletedOnboarding = true` でナビゲーション。callbackチェーンは不安定 |
| WorkoutLogView | elapsed=0でも sheet を表示する（guard条件削除）。E2Eテストでstop直後はelapsed=0になる |
| iOS26+ UserDefaults | `xcrun simctl uninstall` でもUserDefaultsが消去されない。`xcrun simctl spawn $UDID defaults write $BUNDLE_ID key -bool false` で手動リセット必須 |
| Onboarding reset | スクショ撮り直しは uninstall+install だけでは不十分。`defaults write hasCompletedOnboarding -bool false` が必要 |
