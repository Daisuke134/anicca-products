
## Codebase Patterns

| パターン | 詳細 |
|--------|------|
| fastlane test | `LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 fastlane test` 必須（xcpretty UTF-8 バグ回避） |
| xcodegen | ファイル追加後は `xcodegen generate` 必須（テストファイル含め自動pick-up） |
| SubscriptionService env | `@Environment(SubscriptionService.self)` で取得。`SubscriptionService()` 新規生成禁止 |
| URL optional | `URL(string:)!` 禁止。`if let url = URL(string:)` で optional binding |
| Background Timer | `startDate = Date.now` で計時開始。タイマーTick: `Int(Date.now.timeIntervalSince(startDate))`。suspend後も正確 |
| Greenlight | `greenlight preflight <dir>` (xcodeproj じゃなくディレクトリを渡す) |
