**ブロック理由:** `asc apps create` は Apple ID 認証（web session）が必須。iris session expired。

**次のイテレーションで実行:**
```bash
source ~/.config/mobileapp-builder/.env
security unlock-keychain -p "$KEYCHAIN_PASSWORD" ~/Library/Keychains/login.keychain-db
ASC_WEB_PASSWORD="$APPLE_ID_PASSWORD" asc web auth login --apple-id "$APPLE_ID" --two-factor-code <6桁コード>
asc web auth status --apple-id "$APPLE_ID"  # → {"authenticated":true}
ASC_WEB_PASSWORD="$APPLE_ID_PASSWORD" asc apps create --name "LumaRed" --bundle-id "com.aniccafactory.lumared" --sku "lumared" --platform IOS --primary-locale "en-US" --apple-id "$APPLE_ID" --output json
```

**Learnings for future iterations:**
- `asc apps create` は API key 認証 NOT 対応。必ず iris web session が必要
- iris session は 7日程度で expire する。ralph.sh PREFLIGHT Check 6 でセッション確認必須
- 2FA 送信: `ASC_WEB_PASSWORD="..." asc web auth login --apple-id ...` → エラー時に Apple デバイスに 2FA コードが届く
- 次のイテレーション冒頭で `security unlock-keychain` + `asc web auth login --two-factor-code <CODE>` で復帰可能
