## Codebase Patterns

| Pattern | 詳細 |
|---------|------|
| APP_ID in prd.json notes | validate.sh が `APP_ID=(\d+)` regex でprd.json notesから読む。必ずnotesに含める |
| Privacy Policy URL | https://aniccaai.com/privacy を全アプリで再利用。カスタムファイル不要 |
| Project env path | `~/.config/mobileapp-builder/projects/<slug>/.env` (例: desk-stretch) |
| Apify TikTok API | REST API直接呼び出し。`POST /v2/acts/clockworks~tiktok-hashtag-scraper/runs?token=$APIFY_TOKEN` → poll `defaultDatasetId` → `GET /v2/datasets/$ID/items` |
| TikTok result field | `searchHashtag.views` = 該当ハッシュタグの総ビュー数。`playCount` = 個別動画のビュー数 |
| mcpc login not needed | `mcpc --json mcp.apify.com` は認証エラー。REST API直接呼び出しが正解 |
